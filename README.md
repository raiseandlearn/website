# Raise and Learn Website

A React-based website using Vite for Raise and Learn.

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run linting
npm run lint

# Preview production build locally
npm run preview
```

## Deployment

To deploy to GitHub Pages:

```bash
# Build the project
npm run build

# Deploy to GitHub Pages
npm run deploy
```

The site will be deployed to: https://draft.raiseandlearn.com
